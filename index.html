AMC - Annual Maintenance Contract

Documented by Mojoomla - at November 24, 2025

A) Introduction

Annual Maintenance Contract System is ideal way to manage complete Annual Maintenance. It has different user roles like Employee, Customer and admin user.

This user guide covers all options that this react js provides. Please read the documentation carefully as it probably contains answers to most of questions that may arise. If your feel some aspect is not covered or have some questions, contact us via e-mail sales@mojoomla.com. We will do our best to answer your in shortest time.

B) Installation



I) Backend Setup


Require a PHP version ">= 8.2.0".

1) Database Creation


First Create new Database in MySQL.

2) Utilizing Build and Source Packages in React Application


1. Using the Build Package

Purpose: Deploy the application without modifications.

2. Using the Source Package

Purpose: Modify or extend the application.
Setup: Navigate to the source package directory in your terminal.
Installation:Run npm install to install the necessary dependencies.

1) FTP Upload

  1. If you have not FileZilla then download here
  2. Unzip the " amc_package.zip " file
  3. Unzip the " amc_api.zip " file and upload amc_api folder into your server using through FileZilla.
  4. Unzip the " amc_react.zip " file and upload amc_react folder into your server using through FileZilla.

2) Env File Update


Above process completed then go to amc_api/ folder and update .env file in notepad as per the your MySQL database configuration.

Note : MySQL Details Configuration is not required.

For the SMTP Configuration follow this link



II) Frontend Setup


1) Backend URL Update


1. Build Package

Go to amc_react/ folder and change your backend api URL insted of "http://api.amc.niftysol.com/api" this base URL



2. Source Package

Go to amc_react/ folder and change your backend api URL insted of "http://api.amc.niftysol.com/api" this base URL



2) Run the Application


Now Run the Application with react url.
